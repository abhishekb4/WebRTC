# WebRTC
Create WebRTC Library

> Step 1. Add the JitPack maven repository to the list of repositories:

'''gradle
allprojects {
    repositories {
      mavenCentral()
       maven { url "https://jitpack.io" }
        }
   }
> Step 2. Add the dependency
'''gradle
  dependencies {
        implementation 'com.github.User:Repo:Version'
   }
'''
